# Enphase_InfluxDB
Script that converts Enphase local API data to InfluxDB

# Environment Variables
      - ENPHASE_HOST
      - ENPHASE_PASS
      - INFLUXDB_HOST
      - INFLUXDB_PORT
      - DEBUG

# API 
Per Panel
```
            {
                "serialNumber": "122102008624",
                "lastReportDate": 1635382224,
                "devType": 1,
                "lastReportWatts": 0,
                "maxReportWatts": 348
            }
```
